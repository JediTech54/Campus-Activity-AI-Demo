# CampusMatch AI Demo \- Description

CampusMatch AI is an AI‑powered campus activity discovery and matching platform built for university environments\. It connects students with official campus events and peer‑organized activities via intelligent recommendation, personalized ranking, and automated review, solving key pain points including scattered event information, low matching efficiency, and complex manual review management\.

---

## Core Features

1. **AI Personalized Recommendation Engine**

    - Generates smart recommendation scores using students’ ranked interest preferences \(7 categories: sports, career, academic competitions, entertainment, etc\.\), activity type, capacity, and tag matching\.

    - Provides explainable AI recommendation reasons for transparency\.

    - Auto‑generates AI summaries and tags for quick understanding and risk screening\.

2. **Multi‑role \&amp; Full‑cycle Activity Management**

    - Supports three user roles: students \(browse, register, bookmark, customize interests\), official organizers \(publish events\), super administrators \(AI‑assisted review\)\.

    - Manages the full lifecycle: publish, filter, view details, register, bookmark, calendar integration, review\.

3. **Modern UI \&amp; Responsive Interaction**

    - Clean, modern design with gradients, glass‑morphism, and card layout\.

    - Fully responsive for mobile and desktop\.

    - One‑click bookmark/register, drag‑sort interests, calendar view \(month/week\), mobile sidebar\.

4. **AI Review \&amp; Risk Control**

    - Auto‑generates review scores \(0–100\) for peer activities\.

    - Identifies high‑risk content and flags warnings to reduce review workload and compliance risks\.

---

## Value

- **Students**: Reduce information overload, find matching activities easily\.

- **Universities**: Unified event management, AI‑driven safety and compliance\.

- **Organizers**: Reach target participants efficiently, improve attendance\.

---

## Technical Highlights

- Pure front‑end: HTML / CSS / JavaScript

- No backend required, fast demo deployment

- Local data simulation for user state, activities, preferences

- Modular structure, easy to extend \(calendar, notifications, etc\.\)

This demo visually showcases **AI empowering the campus activity ecosystem** with interpretable recommendation and review scoring logic\.
